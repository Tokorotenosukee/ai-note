# AI Notes 運用ルール

このリポジトリは、AIが調査した内容をHTML記事として蓄積する個人用ナレッジベースです。以後の作業では、以下のルールを守ってください。

## 基本方針

- 最初はカテゴリ分けしない
- すべての記事は `notes` フォルダに保存する
- 1記事につき1HTMLファイルとする
- 新しい記事を作成したら `index.html` の最新記事一覧へ追加する
- HTML/CSSのみで構成する
- ビルドツール、フレームワーク、外部ライブラリ、JavaScriptは使わない
- GitHub Pagesでそのまま公開できる構成を保つ
- スマートフォンで読みやすい余白、文字サイズ、行間を優先する

## 記事作成ルール

ユーザーが「質問：」または「記事化：」で始まる指示をした場合：

1. 信頼できる情報源を調査する
2. `notes` に新しいHTML記事を作成する
3. タイトル、作成日、3行まとめ、本文、参考URLを含める
4. `index.html` の最新記事一覧に追加する

## 記事ファイルの書き方

- ファイル名は内容が分かる英数字とハイフンで作成する
- 例: `what-is-ai-notes.html`
- 記事ページでは `../styles/site.css` を読み込む
- 参考URLは実在するURLだけを掲載する
- 参考URLが複数ある場合は、本文末尾に一覧で掲載する
- 調査内容と推測を混同しない

## 禁止事項

- 勝手にカテゴリフォルダを作らない
- 既存記事を大きく構成変更しない
- 出典のない断定を書かない
- 架空のURLを作らない
- APIキーや個人情報を書かない
- JavaScriptを追加しない
- 外部CDNや外部フォントを追加しない
