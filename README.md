# AI Notes

AI Notesは、AIが調査した内容をHTML記事として保存していく個人用ナレッジベースです。

スマホから思いついた疑問をAIに調査させ、あとで読み返しやすい形で `notes` フォルダに蓄積します。ブログではなく、自分専用の「AIが作る辞典」として使うことを想定しています。

## 構成

- `index.html`: トップページと最新記事一覧
- `notes/`: 記事HTMLを保存するフォルダ
- `styles/site.css`: サイト共通のCSS
- `images/`: 画像を保存するフォルダ
- `AGENTS.md`: AIが記事を追加するときの運用ルール
- `.nojekyll`: GitHub PagesでJekyll処理を無効にするためのファイル

## 使い方

1. 新しい調査記事を `notes` フォルダにHTMLファイルとして追加する
2. 記事にはタイトル、作成日、3行まとめ、本文、参考URLを含める
3. `index.html` の `Latest Notes` に新しい記事へのリンクを追加する
4. GitHub Pagesで公開する

## 方針

- HTMLとCSSのみで構成する
- JavaScript、フレームワーク、ビルドツールは使わない
- 最初はカテゴリ分けをせず、すべての記事を `notes` に保存する
- 長期間読み返しやすい、静かな読書ノート風のデザインを保つ
